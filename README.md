# `monte_carlo` Repository

This repository contains codes for various monte carlo simulations. The folders are described below:

1. `MC_PI`: Calculation of value of $\pi$ using simple Monte Carlo method.
2. `VMC_HO`: Simulation of harmonic oscillator using variational Monte Carlo method.
